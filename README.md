# RUNOVA 🏃‍♂️🔥

> **RUNOVA** é um aplicativo web mobile-first de alta performance para acompanhamento de corridas pessoais. Design futurista em **Tema Escuro Profundo**, superfícies **Glassmorphism**, gradientes vibrantes **Laranja** e **Amarelo**, telemetria em tempo real, mapa interativo de rota e gráficos visuais de desempenho.

---

## 🌟 Funcionalidades

- 📱 Interface Mobile-First com navegação thumb-friendly
- ⚡ Telemetria em tempo real (cronômetro, ritmo, velocidade, calorias)
- 📡 Suporte duplo: GPS Real e Modo Simulador
- 🗺️ Mapa interativo de rota em tema escuro (Leaflet + CartoDB Dark Matter)
- 🗣️ Treinador por voz em Português (SpeechSynthesis API)
- ⏱️ Contagem regressiva animada (3... 2... 1... GO!)
- 🫀 Monitoramento de frequência cardíaca (BPM) e cadência (SPM)
- 📊 Gráficos comparativos de desempenho entre sessões
- 📉 Tabela de Splits detalhada por quilômetro
- 🏆 Sistema de conquistas e medalhas desbloqueáveis
- 🌤️ Clima real em tempo real (Open-Meteo API)
- 🔊 Feedback sonoro nativo (Web Audio API)
- 📲 Suporte PWA (instalável como app nativo)
- 💾 Persistência de dados offline via LocalStorage

---

## 🛠️ Tecnologias

- **React 18** + **Vite**
- **Tailwind CSS 3**
- **Lucide React** (ícones)
- **Leaflet** (mapas)
- **Vitest** (testes automatizados)
- **HTML5 Geolocation**, **Web Audio**, **SpeechSynthesis**

---

## 🔒 Privacidade

Toda a telemetria e o histórico são armazenados localmente no dispositivo do usuário via `LocalStorage`. Nenhum dado é enviado a servidores externos.
